# Proyecto Análisis de Ventas

## Integrante
Lucas Vazquez

## Descripción
Este proyecto realiza un análisis de datos de ventas utilizando Python. Se procesan datos desde un archivo CSV para obtener información sobre ventas totales y ventas por producto.

## Estructura del proyecto

- /datos → contiene el dataset utilizado
- /scripts → contiene el script de análisis
- /resultados → contiene el gráfico generado

## Dataset
Archivo CSV con datos de ventas que incluye:
- fecha
- producto
- cantidad
- precio

## Funcionalidades
- Cálculo de ventas totales
- Cálculo de ventas por producto
- Generación de gráfico de barras
